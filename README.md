# 3DGolfGame

## 概要
UnityとC#で制作した3Dゴルフゲームです。  
ボールを操作してひっぱり、できるだけ少ないショット数でゴールを目指します。

## 使用技術
- Unity
- C#

## 実装した機能
- ボールの移動処理
- ゴール判定
- ショット数のカウント
- ステージの変更
- 障害物の設定

## 工夫した点
ボールの操作感が分かりやすくなるように、入力処理とショット数の表示を実装しました。  
また、ゲームの開始からゴールまで一通り遊べる形になるように制作しました。

## 制作目的
UnityとC#の基本操作、物理挙動、シーン内オブジェクトの制御を学ぶために制作しました。
